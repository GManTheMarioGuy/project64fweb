Project64f - Windows Downloads

  • Public Releases >>

    Public releases of the complete Project64 package, for Windows systems. This is the version you most likely want.

  • Development Versions >>

    Development versions are released every time a developer makes a change to Project64, these files are available to patreon supporters. These versions contain the latest code but they are less tested than the public released versions of the emulator.

  • Source Code

    The latest version of the Project64 source code can be downloaded from the project Git repository.

        $ git clone https://github.com/project64/project64.git

    You can also browse the current version of the source code.
